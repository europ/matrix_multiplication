# Matrix Multiplication

Implementation of simple matrix multiplication, including unit tests.

#### Setup

1. `python3 -m pip install virtualenv`
1. `virtualenv .venv`
1. `source .venv/bin/activate`
1. `python3 -m pip install -r requirements.txt`
1. `./mx_mul.py` or `tox`
